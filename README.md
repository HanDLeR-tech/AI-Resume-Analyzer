# 🚀 AI Resume Analyzer & Interview Preparation Platform

An AI-powered full-stack web application that analyzes resumes against job descriptions and provides intelligent, actionable insights to help users improve their chances of getting shortlisted.

The platform takes a user’s **resume, self-description, and target job description**, and evaluates how well the candidate fits the role. It then generates a **compatibility score**, identifies **skill gaps with severity levels**, creates a **personalized interview preparation plan**, and even produces a **customized resume tailored to the job description**.

---

## ✨ Key Features

### 📊 Resume–Job Matching & Scoring

* Analyzes resume against job description
* Generates a **compatibility score (ATS-like score)**
* Helps users understand how suitable they are for a specific role

---

### 🧠 Skill Gap Analysis

* Identifies missing or weak skills
* Categorizes gaps based on **severity (low, medium, high)**
* Provides actionable insights for improvement

---

### 🎯 Personalized Interview Preparation Plan

* Generates a structured roadmap for interview preparation
* Suggests areas to focus on based on job requirements

---

### 💬 Interview Question Generation

* Predicts **technical questions** based on required skills
* Suggests **behavioral questions** based on role expectations
* Helps users prepare for real interview scenarios

---

### 📝 Custom Resume Generation

* Generates a **job-specific optimized resume**
* Tailors content according to job description
* Improves chances of passing ATS filters

---

### 🔐 Authentication System

* User registration and login
* Secure session handling

---

### 📄 Resume Processing

* Supports PDF resume upload
* Extracts and analyzes relevant information

## ⚙️ How It Works

1. User uploads resume (PDF)
2. Enters self-description
3. Provides job description
4. System processes inputs using AI
5. Outputs:

   * Compatibility score
   * Skill gap analysis
   * Interview questions (technical + behavioral)
   * Preparation roadmap
   * Customized resume

---

## 🎯 Use Case

This platform is designed for:

* Students preparing for internships
* Job seekers targeting specific roles
* Anyone looking to improve their resume and interview readiness
